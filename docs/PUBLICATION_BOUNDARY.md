# Publication boundary

This repository is generated from a private game workspace through an explicit
allowlist. Public history must begin here; private repository history must never
be merged, grafted, or force-pushed into this repository.

## Included

- Luau translations/adaptations of GPL Q3 engine gameplay behavior.
- Generic Roblox integration contracts and server-authoritative runtime modules.
- Generic map schema and validation code.
- Build, format, compile, and boundary-verification configuration.

## Excluded

- Authored or reconstructed maps and their measurements.
- Textures, models, meshes, animations, sounds, music, fonts, icons, and video.
- HUD, menus, loading screens, lobby, visual themes, and other client GUI.
- Game-specific VFX and presentation catalogs.
- Store, supporter-pass, currency, inventory-profile, analytics, and publishing code.
- Experience/place/universe/product/pass/asset IDs and credentials.
- Private planning, evidence, staging, upload, and content-generation tooling.

Engine improvements should be exported by allowlist. Never copy the private
workspace wholesale and then attempt to remove content afterward.
